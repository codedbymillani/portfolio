Book Scraper API & Excel Dashboard

Built a dynamic web scraper API that automatically tracks book prices and generates formatted spreadsheet reports.

The Problem

Manually checking websites to track product prices and copying data into Excel spreadsheets is slow, repetitive, and prone to human typos.

My Contributions
  • Coded a flexible web scraper using FastAPI and BeautifulSoup to pull live book titles and prices automatically.
  • Implemented a dynamic query parameter loop so users can change how many pages they want to scrape directly from their web browser.
  • Deployed the live running API to Render, providing a public cloud address and an interactive Swagger documentation dashboard.
  • Programmed a synchronization script with OpenPyXL that pulls the API data and builds an Excel sheet complete with alternating colors, mathematical formulas, and a visual bar chart.
Challenges & Solutions

Hardcoded scraping scripts usually break if you want more data. To solve this, I designed the API to take numbers directly from the web address string, allowing anyone to change the scraping depth on the fly without changing lines of code.
