# Design + Code Portfolio

This is my personal portfolio website, built using clean vanilla front-end code developed in collaboration with Gemini AI. It showcases my work across frontend development, UX/UI engineering, and backend pipelines. I wanted to create a digital space that balances technical logic with soft, clean aesthetics. Instead of leaning on massive corporate frameworks or heavy predefined templates, I paired up with AI to write, debug, and structure lightweight, custom code from scratch.

## Live Site
You can view the live site here: https://codedbymillani.github.io/portfolio/

## The Architecture
The site is divided into dedicated pages to keep the user experience smooth and organized:
* **Home (`index.html`):** A quick snapshot of who I am featuring a dynamic hero section.
* **About (`about.html`):** A closer look at my background, my approach to code and design, and how I bridge the gap between creative layouts and structured systems.
* **Projects (`projects.html`):** An interactive, filterable archive featuring active work and past projects categorized by type.
* **Skills (`skills.html`):** A complete inventory of my technical stack.
* **Contact (`contact.html`):** A direct, simple hub for networking and collaboration.

## Built With
* **HTML5:** Semantic architecture for a solid foundation.
* **CSS3:** Custom layout styles, soft gradients, smooth transitions, and responsive flexbox logic.
* **Vanilla JavaScript:** Light, efficient script logic to manage page behavior and project filtering.
* **FontAwesome:** Minimalist vector iconography for UI accents.
